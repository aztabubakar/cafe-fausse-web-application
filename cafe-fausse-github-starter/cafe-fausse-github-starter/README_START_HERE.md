# Start Here

This repository currently contains the approved project requirements and Day 1 planning material for the Café Fausse web application.

## First Claude Code instruction

Open Claude Code from the repository root and give it this instruction:

> Read `CLAUDE.md` and every file listed inside it. Inspect the repository and compare the Day 1 plan with the SRS. Before writing code, give me: (1) a requirement summary, (2) the proposed directory structure, (3) implementation phases, and (4) missing local prerequisites. Do not change files until I approve the plan.

## Important

- Keep `CLAUDE.md` in the repository root.
- Keep the PDFs and planning files under `docs/`.
- Put the provided restaurant image package under `frontend/src/assets/images/` once available.
- Do not commit `.env` files, passwords, or database credentials.
